# Shell

Edit file `~/.bashrc` or just command in shell

```
export http_proxy=http(socks5)://<proxyAddress>:<port>
export https_proxy=https(socks5)://<proxyAddress>:<port>
export ALL_PROXY=socks5://<proxyAddress>:<port>
```

```
source ~/.bashrc
```
